# SMAART Marketing Command Center

Interactive marketing plan execution platform. Every section of the SMAART marketing plan becomes a clickable, executable segment with modules, best practices, and ad platform integration.

## Quick Start

```bash
git clone https://github.com/smaartdaniel/smaart-command-center.git
cd smaart-command-center
npm install
npm run dev
```

Open **http://localhost:5000** in your browser.

## What You'll See

- **Dashboard** — 16 segments, 64 modules, KPI overview, plan health
- **Segment Pages** — Click any segment to see its modules, best practices, and ad platform tab
- **Creative Playbook** — Interactive Creative Audit Tool (8 sliders), 3×3 Testing Matrix, Ad Templates, Agency Frameworks (Ogilvy, VaynerMedia, Droga5, BBDO, Meta)
- **Dark/Light Mode** — Toggle in the top-right corner

## Tech Stack

- **Frontend:** React + TypeScript + Tailwind CSS + shadcn/ui
- **Backend:** Express + SQLite (Drizzle ORM)
- **Fonts:** Cabinet Grotesk + Satoshi (Fontshare)

## Requirements

- Node.js 18+
- npm 9+
